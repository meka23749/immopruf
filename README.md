# immopruf QA Portfolio
ImmoPrüf — Web Application Quality Assurance Project

A web application for real estate valuation testing.
This project simulates a real QA workflow including manual testing,
automated tests with Playwright, and API testing with Postman.

## Project Structure

- `app/` — Web application (HTML/JS)
- `tests/` — Playwright automated tests (TypeScript)
- `api/` —  REST API (Node.js)
- `docs/` — QA documentation and bug reports

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Testing: Playwright (TypeScript)
- API: Node.js + Express
- API Testing: Postman
- CI: GitHub Actions
